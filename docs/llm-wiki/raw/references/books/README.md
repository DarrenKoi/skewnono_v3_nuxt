# raw/references/books/

책 챕터 노트 (필요 시).

- 파일명: `<book-short-name>-ch<N>.md` (예: `designing-data-intensive-apps-ch3.md`)
- 책 제목·저자·출판년도 표기.
- 챕터별로 분리 권장 (한 책 한 파일이면 길어지고 ingest 어려움).

ingest 시 LLM 이 `wiki/concepts/<topic>.md` 로 합성. 책 한 권의 여러 챕터가 같은 wiki 페이지로 모일 수도 있음.

이 폴더 안 쓰는 팀이면 `git rm -r raw/references/books/` 해도 무방.
