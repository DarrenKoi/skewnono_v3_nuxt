# raw/references/courses/

강의·튜토리얼·온라인 코스 노트.

- 파일명: `<course-name>.md` 또는 `<course-name>-<chapter>.md` (예: `fastapi-official-tutorial.md`, `langchain-academy-rag.md`)
- 코스 URL·플랫폼 표기 (Udemy, Coursera, 공식 docs 등).
- 단계별 노트·코드 스니펫·막힌 지점 포함 OK.

ingest 시 LLM 이:
- 개념 위주면 → `wiki/concepts/<topic>.md`
- 절차 위주면 → `wiki/runbooks/<task>.md`

따라하면서 막힌 지점·본인 reaction 은 `raw/learning-logs/` 에 별도 파일로.
