# raw/references/papers/

논문 요약 (필요 시).

- 파일명: `<year>-<short-title>.md` (예: `2024-rag-survey.md`, `2017-attention-is-all-you-need.md`)
- 형식 권장: 문제·기여·방법·결과·한계 5 섹션 + 본인 적용 아이디어.
- arXiv 또는 DOI URL 필수.

ingest 시 LLM 이 `wiki/concepts/<topic>.md` 의 References 섹션 강한 인용으로 들어간다.

이 폴더 안 쓰는 팀이면 `git rm -r raw/references/papers/` 해도 무방.
