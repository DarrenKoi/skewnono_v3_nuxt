# raw/learning-logs/

"이거 처음 봤다" 거친 학습 메모.

## 파일명

- 권장: `<topic>.md` (예: `langgraph-state.md`, `pydantic-v2-migration.md`)
- 시간 순 회상이 필요한 경우에만 날짜 prefix: `YYYYMMDD-<topic>.md`
- 같은 주제로 여러 번 메모하게 되면 그때 날짜로 구분 (`langgraph-state.md` → `20260501-langgraph-state.md`, `20260615-langgraph-state-deep.md`)
- SCHEMA §2 의 "소문자-하이픈" 만 지키면 OK.

## 내용

- 길이: 3~5 줄도 OK. 완성도보다 빈도가 중요.
- 막혔던 지점, 헷갈린 개념, 검색한 키워드, 참고 링크 등.
- 템플릿: [`../../_templates/learning-log.md`](../../_templates/learning-log.md)

ingest 시 LLM 이 `wiki/concepts/<topic>.md` 로 정제한다.
