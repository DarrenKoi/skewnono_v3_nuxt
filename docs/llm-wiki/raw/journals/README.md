# raw/journals/

개발 저널 — "오늘 무엇을 했고 왜 그렇게 했는가" 의 거친 기록.

- 파일명: `YYYYMMDD-<topic>.md` (예: `20260501-auth-refactor.md`)
- 길이 자유. 5 줄도 되고 한 페이지도 됨.
- 정제하지 말 것. **거친 raw 가 자주 쌓이는 게 정제된 wiki 가 가끔 쌓이는 것보다 가치 있다.**
- 템플릿: [`../../_templates/journal.md`](../../_templates/journal.md)

ingest 시 LLM 이 `wiki/decisions/` 또는 `wiki/components/` 로 합성한다.
