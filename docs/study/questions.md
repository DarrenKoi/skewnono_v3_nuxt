## 질문 형식

아래 템플릿을 복사해 쓰세요. 질문을 번호로 구분하면 답변 파일에서 매칭하기 쉽습니다.

```md
## Q1. <질문 제목>

<본문: 배경, 구체적으로 알고 싶은 점, 원하는 답변 수준>
```

## 예시

## Q1. `$fetch`와 `useFetch`, `useAsyncData`를 언제 무엇을 써야 하나?

사용 맥락이 헷갈린다. 특히 composable 안에서는 `$fetch`만 쓰는 것이 맞는지, page에서는 무조건 `useAsyncData`로 감싸야 하는지 궁금하다.

## Q2. Phase 2로 넘어가면 mock-data 폴더는 삭제해야 하는가?

Phase 1 전용 데이터를 어디까지 유지할지, 테스트용으로 남겨둘지 기준을 세우고 싶다.
